# RouteRus VPN

Быстрый, безопасный и надежный VPN-сервис с обфускацией трафика и удобной административной панелью.

## Особенности

- **Многоуровневая обфускация трафика** с использованием WireGuard, Shadowsocks и Xray
- **Защита от DPI** (Deep Packet Inspection) для обхода блокировок
- **Удобное администрирование** через веб-интерфейс
- **Мониторинг и алерты** с использованием Prometheus и Grafana
- **Уведомления в Telegram** о критических событиях
- **Быстрое развертывание** на новых серверах благодаря Docker

## Архитектура

- **WireGuard**: Основной VPN протокол, обеспечивающий высокую скорость и безопасность
- **Shadowsocks**: Первый уровень обфускации трафика
- **Xray (VLESS)**: Второй уровень обфускации с поддержкой WebSocket поверх TLS
- **FastAPI**: Бэкенд с REST API
- **React**: Административная панель
- **PostgreSQL**: База данных для хранения информации о пользователях и конфигурациях
- **Prometheus + Grafana**: Система мониторинга и визуализации метрик
- **Alertmanager**: Система алертов с интеграцией в Telegram

## Требования

- Ubuntu 20.04+ или другой современный дистрибутив Linux
- Docker и Docker Compose
- Минимум 2 ГБ ОЗУ
- 20+ ГБ дискового пространства

## Быстрый старт

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/anfixit/routerus.git
   cd routerus
   ```

2. Скопировать .env.example в .env и настроить переменные:
   ```bash
   cp .env.example .env
   nano .env
   ```

3. Запустить скрипт установки:
   ```bash
   ./scripts/start.sh
   ```

4. Открыть веб-интерфейс:
   ```
   http://ваш_сервер:8000
   ```

## Мониторинг и логи

### Prometheus и Grafana

Система мониторинга на базе Prometheus собирает метрики о:
- Нагрузке на CPU и RAM
- Использовании диска
- Сетевом трафике (входящем/исходящем)
- Количестве активных VPN-соединений
- Состоянии всех сервисов

Grafana обеспечивает визуализацию этих метрик через красивые дашборды, доступные по адресу:
```
http://ваш_сервер:8000/grafana
```

### Система алертов

Настроена отправка уведомлений в Telegram при следующих событиях:
- Высокая нагрузка на CPU (>80%)
- Критический уровень использования RAM (>80%)
- Недостаток свободного места на диске (<20%)
- Недоступность одного из сервисов
- Подозрительная активность (попытки подбора пароля)

### Логи

Логи всех компонентов системы хранятся в директории `/logs`:
- Логи доступа и ошибок Nginx
- Логи API
- Логи VPN сервисов
- Аудит действий администраторов

## Администрирование

### Веб-интерфейс

Административная панель позволяет:
- Управлять пользователями (создание, удаление, блокировка)
- Создавать и настраивать VPN конфигурации
- Просматривать статистику использования
- Мониторить нагрузку на сервер
- Получать уведомления о критических событиях

### Управление конфигурациями

Для каждого пользователя можно создать:
1. **WireGuard** конфигурации - для максимальной скорости
2. **Shadowsocks** конфигурации - для обхода простых блокировок
3. **Xray (VLESS)** конфигурации - для обхода продвинутых систем DPI

## Безопасность

- Соединения защищены TLS
- Все пароли хранятся в хешированном виде
- Регулярные автоматические обновления
- Защита от брутфорс-атак
- Мониторинг подозрительной активности

## Масштабирование

Система легко масштабируется на дополнительные серверы:

2. Настройте балансировку нагрузки
3. Управляйте всеми серверами через единый интерфейс
